AD5X adapter v7 engineering review

Purpose: gravity-flow/order-fitment only — not pressure-rated plumbing.

Critical gates

  • Female side is on build plate; male thread is not a first-layer surface.
  • 1" female NPT-ish internal thread present: 11.5 TPI, taper, 1.75mm radial depth (~0.8P).
  • 3/4" male NPT-ish external thread present: 14 TPI, taper, 1.45mm radial depth (~0.8P).
  • Internal mesh sampling 0.096mm ≤ 0.12mm layer height.
  • Gradual internal cone avoids previous horizontal/floating transition shelf.
  • FDM approximation only; fit must be tested with real mating fittings before ordering/using anything.

Why v7 exists

v6 was deeper than v5, but the 0.8P NPT thread-depth sanity check showed both thread depths could still be stronger. v7 sets the thread depths near that target while keeping rounded/FDM-printable geometry.
